# skills.md

## 開発スキル

### 物理コア

- 6DoF状態量、座標変換、積分器、空力/重力/推力モデルを分離する。
- まず保守しやすいRK4を使い、必要になれば可変刻み積分器を追加する。
- 単位はSIで統一する。

### 入出力

- CSVはコメント行 `#` を許可する。
- 軌道出力はCSV、Google Earth向けはKMLを生成する。
- 落下分散は将来Monte Carlo化しやすいよう、単発シミュレーション結果を集約できる型を用意する。

### GUI

- GUIはコアライブラリを呼び出すだけにする。
- WindowsではWin32 APIで依存を増やさずローカルオフライン実行を優先する。
- 将来Qt/ImGuiへ移行してもコアには影響させない。

### CI/CD

- GitHub ActionsでWindows/Linuxビルド、テスト、成果物アップロードを行う。
- タグ `v*` がpushされたときはReleaseへ成果物を配置する。
